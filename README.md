# Australia's Wine Insight Analysis - Tableau Dashboard

## Access the Dashboard :- 
https://public.tableau.com/app/profile/jiyansh.garg/viz/Wine_Analysis_Insight/AustraliasWineInsightAnalysis

## Project Overview
This comprehensive Tableau dashboard presents an extensive analytical exploration of Australia's wine industry, providing valuable insights into market trends, regional distribution patterns, and consumer preferences. The interactive visualization serves as a powerful business intelligence tool for stakeholders in the wine sector, enabling data-driven decision-making and strategic planning in one of the world's most prominent wine-producing nations.

### Executive Summary & Key Metrics
- ***Total Reviews Analyzed:*** Comprehensive analysis of 2,305 wine reviews
- ***Geographical Coverage:*** Extensive coverage of Australian wine regions
- ***Quality Assessment:*** Detailed rating distribution across wine varieties
- ***Price Point Analysis:*** Market segmentation by price categories
